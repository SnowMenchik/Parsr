# Parsr
# Установка:
1) Клонируйте репозиторий
2) Установите зависимости:
  bash:
  pip install requests beautifulsoup4 telethon lxml
# Использование:
  Запускайте файл parser.py, далее следуйте инструкции внутри кода
# Получение API-ключей
1) Для VK:
  Получите токен через vkhost.github.io или официальными методами
  Токен должен иметь доступ к методу wall.get
2) Для Telegram:
  Перейдите на my.telegram.org
  Войдите под своим аккаунтом
  Создайте приложение в разделе "API Development Tools"
3) Для OK:
  Перейдите на dev.odnoklassniki.ru
  Зарегистрируйтесь как разработчик (требуются паспортные данные)
  Создайте Standalone-приложение
